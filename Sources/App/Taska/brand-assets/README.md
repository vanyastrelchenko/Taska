# Brand Assets

Папка для хранения бренд-ресурсов Taska.

## Структура

```
brand-assets/
├── icons/          # SVG иконки для веб/маркетинга
├── logo/           # Логотип (монограмма + словознак, светлая/тёмная версии)
└── README.md       # Этот файл
```

## Спецификация

### Иконки (SVG)
- Размер сетки: `24×24 px`
- Толщина stroke: `≈1.5 px`
- Формат: SVG

### Логотип
- Форматы: SVG, PNG (для различных применений)
- Версии: светлая, тёмная
- Варианты: монограмма, словознак, полный логотип

## Использование

В iOS приложении используются нативные SF Symbols. SVG иконки предназначены для:
- Веб-версии приложения
- Маркетинговых материалов
- Документации

Подробная документация бренд-токенов: [`docs/STYLEGUIDE.md`](../docs/STYLEGUIDE.md)
