# Contributing to Taska

## Ветвление
- `main` защищена, изменения — через PR.
- Фичи: `feature/<slug>`, чоры: `chore/<slug>`, фиксы: `fix/<slug>`.

## PR
- Малые и атомарные изменения.
- Шаблон PR обязателен; скриншоты приветствуются.
- CI должен быть зелёным. Мёрдж — Squash.

## Коммиты
- Стиль: `feat:`, `fix:`, `chore:`, `docs:`, `ci:`, `refactor:`.
